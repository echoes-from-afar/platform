# Code of Conduct

## Our Commitment

Echoes from Afar is a care-centered archive. We are committed to a respectful, inclusive, and harassment-free experience for everyone participating in this project and community.

## Expected Behavior

- Use respectful and inclusive language.
- Assume good intent, but listen when impact is harmful.
- Protect contributor privacy and consent.
- Engage with curiosity, humility, and accountability.

## Unacceptable Behavior

- Harassment, discrimination, hate speech, or demeaning language.
- Doxxing or sharing private information without explicit consent.
- Threats, intimidation, or sustained disruption of conversations.
- Content or conduct that pressures others to disclose personal trauma.

## Scope

This Code of Conduct applies to:

- GitHub issues, pull requests, and discussions.
- Project documentation and review comments.
- Synchronous and asynchronous project communication channels.

## Reporting

If you experience or witness harmful behavior:

- Open a private report to maintainers through a GitHub issue marked `confidential` (or request one via maintainers in an existing issue).
- For security-sensitive incidents, follow [SECURITY.md](./SECURITY.md).

Maintainers will review reports promptly, keep details private where possible, and communicate decisions clearly.

## Enforcement

Maintainers may take any action they deem appropriate, including warning, temporary or permanent removal from community spaces, or rejecting contributions.
